# FiSiQuiz App 🚀

Deine persönliche Lern-App für die IHK-Abschlussprüfung (Fachinformatiker/in).

## Über das Projekt
Diese App hilft dir, dich gezielt auf die Wirtschafts- und Sozialkunde (WiSo) Prüfungen vorzubereiten.
- **Features:** Multiple-Choice & Single-Choice, detaillierte Lösungswege (besonders bei Rechenaufgaben!), Fortschrittsanzeige und Ergebnis-Dashboard.
- **Technologie:** Swift, SwiftUI, JSON-Datenbank.

## Struktur
- `fragen.json`: Enthält alle IHK-Prüfungsfragen und Lösungswege.
- `ContentView.swift`: Das UI für das Quiz.
- `DataLoader.swift`: Die Logik zum Einlesen der Fragen.
